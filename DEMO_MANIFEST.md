# Bounty-Hive Demo Manifest (demo-v1)

## Status
This repository is **frozen** as of demo-v1.

No functional, behavioral, or content changes will be made.

## Purpose
This demo exists solely to demonstrate:

- Governance boundaries
- Trust separation between Core and Engine
- Disclosure lifecycle guarantees
- Audit-first design principles

It does **not** perform scanning, exploitation, or analysis.

## What This Demo Includes
- Repository-level freeze notice
- Governance documentation
- Disclosure lifecycle modeling
- Trust-boundary declarations

## What This Demo Does NOT Include
- Autonomous scanning
- Vulnerability discovery
- Exploit logic
- Proof-of-concept payloads
- Engine internals
- Customer data handling

## Relationship to Production System
This demo is a **structural reference artifact**, not a production build.

Production capabilities exist only in:
- Controlled environments
- Explicit enterprise authorization
- Paid, gated execution paths

## Immutability Statement
demo-v1 is an immutable reference point.

Any future demos will be versioned separately and explicitly labeled.
